##	简介

*	使用逻辑回归进行预测

*	特征工程

	与特征工程相关的文件放在了文件夹 `feature_analysis` 中

	这里使用了 `Pclass`, `Sex`, `Age`, `Fare`, `Embarked`, `NameLength`, `HasCabin`, `Title`, `IsChildren`, `FamilySize`, `IsAlone` 作为输入特征

*	代码放在了 `src` 文件夹下

*	结果放在了 `result` 文件夹下
